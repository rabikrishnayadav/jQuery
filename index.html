Learn jQuery

Introduction

.jQuery is a lightweight, 'write less, do more', JavaScript library.
.Liberary containing a set of functions(reusable functions) created using JavaScript.

Why jQuery?

.jQuery takes a lot of common tasks that require many lines of JavaScript code to accomplish, and wraps them into methods that you can call with a single line of code.
.jQuery also simplifies a lot of the complicated things from JavaScript, like Ajax calls and DOM manipulation. Also from css to ex. css manipulation, Effects and animations
lQuery functions are completely browser independent.
jQuery Syntax
Basic syntax is:
$(selector).action()
Ex. $('#test').hide();
Where,
A $ sign to define/access jQuery
A (selector) to 'query(or find)' HTML elements
A action() to be performed on the elements.

How to use jQuery

There are two versions of jQuery available for downloading:
.Production Version - this is for live website because it have been minified and compressed.
.Development Version - this is for testing and development(uncompressed and readable code).

jQuery Selector

jQuery selectors are used to find (or select) HTML elements based on their name,id,classes,type,attributes,values of attibutes and much more.

Syntax:
For Element:
$('h6') // this selector will select the html h6 tag elements.
$('#hidebtnid') // this selector will select the element id name
$('.hidetxtclass') // this selector will select the element by class name
$('div:first') // this selector will select first div of the whole document
$('div:last') // this selector will select last div of the whole document
 Result:
This paragraph will hide by using it's element name click on hide
